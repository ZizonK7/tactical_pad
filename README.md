## 🇰🇷 업데이트 소식 (v1.1.0)

새로운 기능이 추가된 **v1.1.0** 버전을 배포합니다! 이번 업데이트에서는 사용자 편의성을 높이기 위한 드로잉 도구와 환경 설정 기능이 개선되었습니다.

### 🚀 주요 변경 사항
* **선 그리기 기능 추가**: 이제 자유로운 선 그리기를 통해 더 세밀한 표현이 가능합니다.
* **전체 삭제 기능 추가**: 클릭 한 번으로 캔버스 위의 모든 요소를 즉시 초기화할 수 있습니다.
* **창 크기 조절 지원**: 사용자의 작업 환경에 맞춰 프로그램 창 크기를 자유롭게 변경할 수 있습니다.

## 🇺🇸 What's New (v1.1.0)

We are excited to announce the release of **v1.1.0**! This update focuses on enhancing user experience with new drawing tools and improved flexibility.

### 🚀 Key Features
* **Line Drawing Tool**: You can now draw free-form lines for more detailed work.
* **Clear All Function**: Added a feature to reset the entire canvas with a single action.
* **Resizable Window**: The application window can now be resized to fit your workspace perfectly.
